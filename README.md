# 🚗 Driver Drowsiness Detection System

## 📌 Description
This project aims to detect driver drowsiness in real-time using computer vision techniques. It helps prevent accidents by monitoring eye closure and yawning patterns.

---

## 🎯 Objective
To build a system that alerts drivers when signs of drowsiness are detected.

---

## 🧠 Technologies Used
- Python
- OpenCV
- NumPy

---

## ⚙️ Working Principle
- Captures real-time video using webcam
- Detects face, eyes, and mouth using Haar Cascade classifiers
- Tracks eye closure and yawning duration
- Generates drowsiness score
- Displays alert if driver is drowsy
